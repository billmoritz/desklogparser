# desklogparser

## Requires
* Ruby 2.2

## Install
```
gem install bundler
bundle install
```

## Usage: 
```
bin/logparser [options]

Options:
    -f, --file LOGFILE               Specify the LOGFILE to parse
        --type [TYPE]                Select the logfile format TYPE (common, combined)
    -h, --help                       Show this message
        --version                    Show version
```
